# Abdullah Hamdan Expense/Budget tracker react

## Description
A responsive React application for managing personal finances with intuitive budget tracking. 

## Features
1) **Budget Management: Create, view, and delete multiple budgets with individual spending limits**

2) **Expense Tracking: Add expenses to specific budgets with name, amount, and date**

3) **Visual Progress Indicators: Color-coded progress bars showing spending relative to budget limits**

4) **Responsive Design: Works seamlessly on desktop, tablet, and mobile devices**

5) **Local Storage: All data persists between browser sessions**

6) **Real time Calculations: Automatic calculation of remaining budget amounts**

7) **Expense Filtering: View expenses filtered by specific budgets**

## Techonlogies

1) **React: Frontend library for building user interfaces**

2) **React Bootstrap: UI component library for responsive design**

3) **Bootstrap CSS framework for styling and responsiveness**

4) **Local Storage**

5) **React local hook**

## Run

**npm run start**
